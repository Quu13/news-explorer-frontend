# 📰 News Explorer

## 📌 Project Description

This project is a responsive news application built with **React**. It fetches and displays a variety of current news articles using a **third-party News API**, allowing users to search by keyword. The app includes a clean user interface, keyword-based search functionality, and simulated authentication for saving articles.

## 🌐 Live Demo

🔗 [Deployed Application](https://Quu13.github.io/news-explorer-frontend)

## 🛠️ Technologies Used

- **React**
- **JavaScript (ES6+)**
- **HTML5**
- **CSS3**
- **Vite**
- **React Router**
- **NewsAPI** (for fetching articles)
- **Git & GitHub Pages** (for deployment)
